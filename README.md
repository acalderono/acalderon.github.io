##  Stack

React, Styled-Components, Tailwindcss

  
## Environment Variables

Para ejecutar este proyecto, deberá agregar las siguientes variables de entorno a su archivo .env

`REACT_APP_TOKEN_GITHUB`

  
## Run Locally

Instalar dependencias

```bash
  npm install
```

Inicie el servidor

```bash
  npm run start
```
